  

Georeferencing Your Historic Map
    1. Go to mapwarper.net and create an account
    2. Download the Paramaribo 1916-1917 Map from resources/txu-oclc-51897369-paramaribo-1941.jpg
    3. Change the filename to add your initials to the beginning of the name
    3. Use 'Upload Map' tab to upload the Paramaribo map (if you are doing this with another map you should make sure that providing the map publicaly does not violate copyright)
    5. Within the Map page click on the 'Rectify' tab
    6. Using the pan and zoom tools, find a place that you can identify on both maps
    7. Using the pen tool, click on the same place on both maps and then click the 'Add Control Point' button
    8. Repeat for a total of at least three control points
    9. Click 'Warp Image!'
    10. In the 'Export' tab make note of the number that you find in the string like this (20825 in this example):
        http://mapwarper.net/maps/tile/20825/{z}/{x}/{y}.png
        This will be your link to the image tiles


Creating Vector Data

    Bringing Your Historic Map into GeoJSON.io
        1. Go to geojson.io
        2. From the Meta menu item, select Add map layer, and paste in your mapwarper tiles string
        3. Zoom into Paramaribo and confirm that you can see the map you georeferenced, overlaid on the city

    Tracing
        1. Using the tools on the right side of the map you can add polygons, rectangles, points or lines
            Double click to finish a polygon or line
            You can use the table view to add attribute information about any of your vectors
            You can also click on a shape to open the attribute editor for that individual item
        2. 

    Importing
        1. Simply drag your csv (comma seperated value--excel and google docs can export .csv files) to the map
        As long as the file has a Latitude and Longitude column it will add them as points to map
        #note that geojson.io requires values for each latitude and longitude otherwise it will fail (set to 0,0 if you do not know)


    Exporting
        Save out your geojson by using the 'Save' menu and selecting geojson, we will then paste it into our javascript below

    javascript:
        // use a variable for the 1916 tile set
        var paramaribo_1916 = L.tileLayer('http://mapwarper.net/maps/tile/8120/{z}/{x}/{y}.png');
        var paramaribo_mod = L.tileLayer('http://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
            attribution: '&copy; <a href="http://www.openstreetmap.org/copyright">OpenStreetMap</a>'
        });

        // create the map with the tilesets
        var map = L.map('map', {
            layers: [paramaribo_mod, paramaribo_1920]
        });

        // zoom and center in Ann Arbor
        map.setView([5.668, -55.2], 14);


        var geo_json = 'put your geojson here';

        var geodata = JSON.parse(geo_json);

        //we'll change this once we add the popup
        var geolayer = L.geoJson(geodata);

        // add the points to the map
        geolayer.addTo(map);





    Javascript function to add a popup:

        function showPopup(feature, layer) {
            var key, val;
            var content = [];
            for (key in feature.properties) {
                val = feature.properties[key];
                if (key == "Image URL") {
                    val = '<img src="' + val + '"  width="150" />';
                }
                content.push("<strong>" + key + ":</strong> " + val);
            }
            layer.bindPopup(content.join("<br />"));
        }

    Change the command to add the layer to:

        var geolayer = L.geoJson(geodata, {
            onEachFeature: showPopup
        });
